knowhow/xtuple paket "u2"
====================================

Info
----

Ovaj paket sadrži utility funkcije potrebne za inteligentniji update drugih paketa

Instalacija
------------

Pakovanje se vrši na sljedeći način: 

<pre>
$ cd package
$ ./make_pkg.sh
</pre>

Dobijeni .gz sa updater-om instaliramo na knowhow/xtuple.
